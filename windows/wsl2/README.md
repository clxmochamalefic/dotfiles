# README.md - WSL2

## use WSL2

### install

```pwsh
wsl --install
```

### set version (WSL2)

```pwsh
wsl --set-version 2
```

### find official distribution

```pwsh
wsl --list --online   
```

### install official distribution

> e.g. Ubuntu

```pwsh
wsl --install Ubuntu-24.04   
```
