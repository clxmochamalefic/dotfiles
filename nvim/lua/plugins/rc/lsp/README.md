# README - lua.plugins.rc.lsp

## 設定の作成ルール

### 各 LSP/DAP/LINTER/FORMATTER別の定義ルール

それぞれのディレクトリの中に `config` ディレクトリを作成し、その中に `init.lua` を作成する

`init.lua` の中で各LSP別の設定を `require()`  をコールする

#### 型定義

私はよく適当に設定の構造を作成しがちで本当に良くないので、各 `config/sub` ディレクトリの中に `_types.lua` を作成し、そこに設定別の型定義を作成する

その型定義をインターフェースとし、そこから自動で設定を展開できるようにする

#### ユーティリティ

必要に応じてユーティリティとして `_utils.lua` を `config/sub` ディレクトリの中に配置してもよい
